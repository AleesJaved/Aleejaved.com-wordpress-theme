# Aleejaved.com WordPress Theme
 
A modern, customizable WordPress portfolio theme with extensive customization options and interactive features.  
I made this for my own website but have made it customisable for everyone to use!  
Check out what it looks like: www.aleejaved.com
 
## 🚀 Quick Installation
 
1. **Download** the latest [release](https://github.com/AleesJaved/Aleejaved.com-wordpress-theme/releases)
2. **Upload** the [aleejaved-portfolio](cci:9://file:///Users/aleejaved/windsurf-portfolio-wordpress/wp-content/themes/aleejaved-portfolio:0:0-0:0) folder to `/wp-content/themes/`
3. **Activate** the theme in WordPress admin under **Appearance → Themes**
4. **Customize** via **Appearance → Customize**
 
## ✨ Key Features
 
- 🎨 **Full Color Customization** - Accent, background, text, muted text, and borders
- 🦸 **Hero Section** - Customizable greeting, name, and subtitle
- 🔗 **Dynamic Buttons** - Add/remove portfolio buttons with custom links
- 📅 **Timeline System** - Sortable timeline with impressiveness ratings
- 🎂 **Birthday Cards** - Interactive card creation with animations
- 💬 **Popup System** - Custom popup management
- 📱 **Page Order Management** - Drag-and-drop page ordering
- 📐 **Responsive Design** - Mobile-friendly layout
